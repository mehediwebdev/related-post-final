== Description ==

The WP Post Information plugin is a simple WordPress plugin designed specifically to display post information such as total words, characters, and approximate reading time.

  
 **How does it work?**
== To enable the settings, navigate to Settings => WP Post Information and select the settings option there.

== Check the output from the single post page to see if it appears before or after the post content.

== Key Feature's  ==
 
 * It displays the number of words, characters, and the approximate reading time of the post."
